# Git 충돌 해결하기 워크숍

따로 작업한 작업물을 머지할 때, 같은 파일에 같은 부분을 수정했다면 충돌이 발생합니다.
이러한 충돌은 자동으로 해결할 수 없으며 충돌이 발생한 작업을 한 사람들끼리 직접 해결해야 합니다.
이러한 충돌이 발생했을 때 어떻게 해결해야 하는지 알아봅니다.

## 목표

* Git 충돌이 언제 일어나는지 안다.
* Git 충돌을 해결할 수 있다.
